Questions Assignment 1

  1. What does the server in the client server architecture do?
    The server hosts the webpage (the HTML files) and deliver them to the client upon request.
  2. What does the client do?
    The client requests a file (the webpage) from the server and renderer the webpage in a web browser when it recieves the files.
  3. What is an IP address?
    An IP address is an adress used in a network such that packages are sent to the correct computers.
  4. What is DNS?
    DNS stands for Domain Name Server and is responsible of translating human readable urls like www.google.com to the correct ip address (216.58.211.132).
  5. What is the difference between the Internet and WWW?
    The Internet is the physical network while WWW is an application unning on the Internet
  6. Explain what HTML is
    HTML is a markup language that web browsers can read to undersand what the content of a webpage is.

Questions Assignment 2

  1. Why does inline style CSS override rules defined in style elements and external stylesheets?
    The last style defined is applied. Since the .css stylesheet is loaded at the beginning of the html file, all styles defined later in the file will overwrite the .css file.
  2. Give a brief example of when to use ID (#id) and when to use classes (.class) in CSS.
    ID is used to select one unique element in the HTML file. The classes selects elements that are part of a perticular class.
  3. What does RGBA mean and what colors can you express with it?
    RGBA colors are the normal RGB colors but also specify opacity.
